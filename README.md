## Скрипт генерации и анализа школьного журнала

### Описание

Этот проект представляет собой Python-скрипт для генерации случайного
школьного журнала и анализа успеваемости учеников.

Скрипт: - Генерирует имена учеников с помощью библиотеки Faker (локаль
ru_RU) - Создаёт оценки по трём предметам - Рассчитывает средний балл -
Выводит статистику по классам — Показывает топ учеников по среднему
баллу

Репозиторий проекта:\
https://github.com/Qmndrn/MatStforDS_1.git

Запуск скрипта:

    python main.py

------------------------------------------------------------------------

### Что делает скрипт

-   Создаёт два класса: "А" и "Б"
-   Генерирует по 10 учеников в каждом классе
-   Назначает случайные оценки от 3 до 5 по предметам:
    -   matem (математика)
    -   rus (русский язык)
    -   inform (информатика)
-   Вычисляет средний балл каждого ученика
-   Формирует:
    -   статистику по количеству оценок в разрезе классов
    -   топ 3 учеников по среднему баллу
